Ember API Herramientas Filosofía y diseño

Según la empresa, Ember se diseñó en torno a cuatro ideas clave:

Concéntrese en aplicaciones web ambiciosas

Ember se propone brindar una solución integral al problema de la aplicación del lado del cliente.

Mayor productividad desde el primer momento

Ember es un componente de un conjunto de herramientas que funcionan juntas para proporcionar una pila de desarrollo.

Estabilidad sin estancamiento

Esta es la idea de que la compatibilidad con versiones anteriores es importante y se puede mantener mientras se sigue innovando y evolucionando el framework.

Previsión de futuros estándares web

Ember adoptó los estándares en torno a JavaScript y la web, incluidas las promesas, los componentes web y la sintaxis de ES6.
